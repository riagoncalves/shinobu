## Installation

### Requirements

-   Node v12.0.0 or higher
-   A Discord developer account

### Getting Started

-   Clone this repo and run `yarn install`
-   Create a copy of `.env.example` named `.env`
-   Fill in with your env variables
-   Run `yarn start` to start
